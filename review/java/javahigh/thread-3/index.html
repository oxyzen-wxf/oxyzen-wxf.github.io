# java8 锁的优势

如果使用 synchronized 锁，通过 notify 去唤醒按顺序线程指定任务比较困难，需要提升线程的优先级。但是 java8 的 Lock 锁可以实现精准打击，可以通过一把锁配多把钥匙的方式简单完成按固定顺去执行。
